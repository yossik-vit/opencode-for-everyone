# Загрузка и запуск проекта

## Загрузка файлов курса

### Через Git (рекомендуется)

```bash
git clone https://github.com/your-org/opencode-course.git
cd opencode-course
```

### Через ZIP

Скачайте архив курса и распакуйте:

```bash
unzip opencode-course.zip
cd opencode-course
```

## Структура проекта курса

```
opencode-course/
├── module-0/           # Установка и настройка
├── module-1/           # Основы работы
├── module-2/           # Практический проект
├── exercises/          # Упражнения
├── solutions/          # Решения
├── AGENTS.md          # Инструкции для OpenCode
└── README.md          # Описание курса
```

## Запуск OpenCode в проекте курса

```bash
# Перейдите в директорию курса
cd opencode-course

# Запустите OpenCode
opencode
```

## Что такое AGENTS.md?

Файл `AGENTS.md` — инструкция для OpenCode о вашем проекте. Работает как память ассистента.

### Пример AGENTS.md

```markdown
# Инструкции для OpenCode

## О проекте
Это обучающий курс по работе с OpenCode.

## Соглашения
- Используй TypeScript
- Пиши тесты для всех функций
- Следуй ESLint конфигурации

## Команды
- `npm test` — запуск тестов
- `npm run lint` — проверка стиля кода
- `npm run build` — сборка проекта

## Структура
- `/src` — исходный код
- `/tests` — тесты
- `/docs` — документация
```

## Переключение агентов

Нажмите **Tab** для переключения между режимами:

- **Сборка** — для генерации кода
- **Планирование** — для планирования и анализа

## Полезные команды терминала

```bash
# Запуск с конкретной моделью
opencode --model deepseek/deepseek-r1

# Запуск с конфигурацией из файла
opencode --config ./project.opencode.json

# Режим интерактивного чата
opencode chat

# Выполнение одиночного запроса
opencode run "объясни этот код" --file src/index.ts
```

## Настройки проекта

Создайте `.opencode/` директорию для локальных настроек:

```bash
mkdir .opencode
echo '{"model": "deepseek/deepseek-r1"}' > .opencode/config.json
```
