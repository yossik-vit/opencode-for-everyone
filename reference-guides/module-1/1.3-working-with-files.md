## Создание файлов

### Новый файл

```
> Создай файл src/components/Button.tsx с базовым компонентом кнопки
```

AI-агент создаст файл с правильной структурой:

```tsx
// src/components/Button.tsx
import React from 'react';

interface ButtonProps {
  label: string;
  onClick: () => void;
  variant?: 'primary' | 'secondary';
}

export const Button: React.FC<ButtonProps> = ({ 
  label, 
  onClick, 
  variant = 'primary' 
}) => {
  return (
    <button 
      className={`btn btn-${variant}`}
      onClick={onClick}
    >
      {label}
    </button>
  );
};
```

## Безопасность работы с файлами

### Что AI-агент делает автоматически
- Создаёт бэкапы перед изменениями
- Показывает diff перед сохранением
- Проверяет синтаксис после изменений

### Что вы можете сделать
- Проверяйте git status перед коммитом
- Используйте git stash для временных изменений
- Регулярно делайте коммиты
