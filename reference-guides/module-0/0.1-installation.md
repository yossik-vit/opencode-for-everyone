# Установка Claude Code

## Системные требования

- Node.js 18+ или Go 1.21+
- macOS, Linux или Windows (WSL)
- Терминал с UTF-8

## Способ 1: Через npm (рекомендуется)

```bash
npm install -g @anthropic-ai/claude-code
```

## Способ 2: Через Homebrew (macOS)

```bash
brew install anthropics/tap/claude-code
```

## Способ 3: Через Go

```bash
npm install -g @anthropic-ai/claude-code@latest
```

## Способ 4: Десктопное приложение

Скачайте установщик с официального сайта:
- macOS (Apple Silicon / Intel)
- Windows
- Linux (AppImage / deb)

## Проверка установки

```bash
claude --version
```

## Первоначальная настройка

### Вариант A: Бесплатные модели

Claude Code включает бесплатные модели:

```bash
claude init
```

Ответьте "да" на вопрос об использовании бесплатных моделей.

### Вариант B: API ключи

```bash
# Установите ключ для выбранного провайдера
export OPENAI_API_KEY="sk-..."
export ANTHROPIC_API_KEY="sk-ant-..."
# или любой другой провайдер из 75+ доступных
```

### Настройка через конфигурационный файл

Создайте или отредактируйте `~/.claude/settings.json`:

```json
{
  "provider": "openai",
  "model": "gpt-4o",
  "providers": {
    "anthropic": {
      "apiKey": "sk-ant-..."
    },
    "openai": {
      "apiKey": "sk-..."
    }
  }
}
```

## Установка плагина для редактора (опционально)

Claude Code поддерживает интеграцию с популярными редакторами:

### VS Code
```bash
code --install-extension anthropic.claude-code
```

### Neovim (через LSP)
Добавьте в конфигурацию Neovim поддержку встроенного LSP Claude Code.

## Устранение проблем

### Проблема: Команда не найдена
Решение: Убедитесь, что путь к глобальным npm пакетам в PATH:
```bash
export PATH="$(npm root -g):$PATH"
```

### Проблема: Ошибки авторизации
Решение: Проверьте API ключи и их формат в `~/.claude/settings.json`.

### Проблема: Тихий вывод
Решение: Проверьте настройки терминала и кодировку UTF-8.
