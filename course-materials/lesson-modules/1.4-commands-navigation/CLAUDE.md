## Think ключевые слова

**Важно:** Эта фича работает только с моделями Claude (Opus, Sonnet, Haiku). В других моделях (GLM, GPT, Gemini) эти команды не работают — модель думает на своём уровне по умолчанию.

Если у тебя Claude, можешь управлять глубиной мышления:
- "think about X" — нормальное мышление
- "think harder" — глубокий анализ
- "ultrathink" — максимальная мощность (увидишь радугу!)

Используй для сложного — стратегия, аналитика, хитрые проблемы. Но глубокое мышление расходует лимиты.

Если у тебя не Claude — не страшно. Твоя модель всё равно думает, просто без управления глубиной. Для сложных задач просто проси: "подумай глубже и дай развёрнутый анализ".

## Три режима ввода

Три режима:

- **Edit mode** — по умолчанию. Спрашиваю разрешение. Безопасно, медленно
- **Auto-accept mode** — просто делаю. Быстро, но доверяешь мне
- **Plan mode** — план, проверка, выполнение

Переключение: Shift+Tab

Edit или Auto подходит. Plan для сложных задач. Можешь обсудить план в чате.

## Финал: Режим без подтверждений

**Важно:** Эта фича специфична для Claude Code (`claude --dangerously-skip-permissions`). В opencode и других AI-клиентах может не быть аналога — или он работает по-другому.

Что это делает: убирает все запросы на подтверждение. AI просто делает всё без вопросов.

⚠️ Опасно: полный контроль, никаких проверок. Но если доверяешь — это самый быстрый способ работы.

Если у тебя Claude Code — запуск: `claude --dangerously-skip-permissions`
Если у тебя opencode или другой клиент — проверь документацию, есть ли аналог. В opencode по умолчанию меньше запросов на подтверждение.

Опытные пользователи со временем переходят на режим без подтверждений. Когда почувствуешь себя комфортно — попробуй. Но не с первого дня.

## Важные заметки для OpenCode

- **Демонстрация Escape**: Начни прерываемое, практикуй Escape
- **Демонстрация отката**: Попробуйте Esc×2
- **Плановый режим**: Не продавай сильно. Полезен, но не критичен
- **Dangerously skip permissions**: Для доверенных процессов, не для всего

## Особенности OpenCode

### Модели
75+ провайдеров:
- Claude (Opus, Sonnet, Haiku)
- GPT (GPT-4, GPT-4o, O1, O3)
- Gemini (2.5, 2.0 Flash)
- Локальные модели (Ollama, LM Studio)

### Агенты
- **build** — полный доступ для разработки
- **plan** — read-only для анализа и исследования

Переключение: `Tab`

### Конфигурация
Настройки хранятся в `~/.opencode.json`

### Desktop
Desktop: macOS, Windows, Linux (бета)

### Клавиатурные сокращения
- `Ctrl+C` — выход
- `Ctrl+?` — справка
- `Ctrl+A` — переключение сессий
- `Ctrl+K` — командный диалог
- `Ctrl+O` — выбор модели
- `Esc` — закрыть диалог
