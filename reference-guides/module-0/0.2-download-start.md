# Загрузка и запуск проекта

## Загрузка файлов курса

### Через Git (рекомендуется)

```bash
git clone https://github.com/your-org/ai-trainer.git
cd ai-trainer
```

### Через ZIP

Скачайте архив курса и распакуйте:

```bash
unzip ai-trainer.zip
cd ai-trainer
```

## Структура проекта курса

```
ai-trainer/
├── module-0/           # Установка и настройка
├── module-1/           # Основы работы
├── module-2/           # Практический проект
├── exercises/          # Упражнения
├── solutions/          # Решения
├── AGENTS.md          # Инструкции для Claude Code
└── README.md          # Описание курса
```

## Запуск Claude Code в проекте курса

```bash
# Перейдите в директорию курса
cd ai-trainer

# Запустите Claude Code
claude
```

## Что такое AGENTS.md?

Файл `AGENTS.md` — инструкция для Claude Code о вашем проекте. Работает как память ассистента.

### Пример AGENTS.md

```markdown
# Инструкции для Claude Code

## О проекте
Это обучающий курс по работе с Claude Code.

## Соглашения
- Используй TypeScript
- Пиши тесты для всех функций
- Следуй ESLint конфигурации

## Команды
- `npm test` — запуск тестов
- `npm run lint` — проверка стиля кода
- `npm run build` — сборка проекта

## Структура
- `/src` — исходный код
- `/tests` — тесты
- `/docs` — документация
```

## Переключение агентов

Нажмите **Tab** для переключения между режимами:

- **Сборка** — для генерации кода
- **Планирование** — для планирования и анализа

## Полезные команды терминала

```bash
# Запуск с конкретной моделью
claude --model deepseek/deepseek-r1

# Запуск с конфигурацией из файла
claude --config ./project.claude.json

# Режим интерактивного чата
claude chat

# Выполнение одиночного запроса
claude run "объясни этот код" --file src/index.ts
```

## Настройки проекта

Создайте `.claude/` директорию для локальных настроек:

```bash
mkdir .claude
echo '{"model": "deepseek/deepseek-r1"}' > .claude/config.json
```
