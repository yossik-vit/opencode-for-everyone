# Рабочее пространство Claude Code

## Интерфейс терминала

Claude Code работает в терминале, интегрируясь в рабочий процесс.

```
╭──────────────────────────────────────────────────────────╮
│  Claude Code                              Ваша модель  🔨   │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  > Ваш вопрос или задача...                              │
│                                                          │
├──────────────────────────────────────────────────────────┤
│  ⏎ отправить  Tab переключить  Ctrl+C выйти  / помошь   │
╰──────────────────────────────────────────────────────────╯
```

## Элементы интерфейса

### Заголовок
- Название: Claude Code
- Текущая модель (выбирается через Ctrl+O)
- Текущий режим (🔨 сборка или 📋 планирование)

### Область ввода
- Строка для ввода запросов
- Поддержка многострочного ввода (Shift+Enter)

### Статусная строка
- ⏎ — отправить сообщение
- Tab — переключить режим агента
- Ctrl+C — выйти
- / — показать команды

## Режимы агентов

### Режим сборки (🔨)
Генерация и изменение кода:
- Создание новых файлов
- Редактирование кода
- Запуск команд
- Работа с Git

### Режим планирования (📋)
Анализ и планирование:
- Анализ кодовой базы
- Планирование архитектуры
- Объяснение кода
- Рекомендации

### Переключение режимов
Нажмите **Tab** для переключения между режимами. Claude Code запомнит выбор.

## Советы по эффективности

1. **Используйте Tab** — быстрое переключение режимов экономит время
2. **Задавайте контекст** — чем детальнее запрос, тем лучше результат
3. **Используйте /** — для просмотра всех доступных команд
4. **Читайте ответы** — Claude Code объясняет свои решения
5. **Экспериментируйте** — пробуйте разные формулировки запросов
