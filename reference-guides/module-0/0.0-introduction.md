# Введение в курс "OpenCode для всех"

## Что такое OpenCode?

OpenCode — это AI-ассистент для разработки в терминале, созданный для работы с вашим кодом и проектами. Он интегрируется в ваш рабочий процесс, помогая писать код, исправлять ошибки, создавать документацию и многое другое.

## Ключевые преимущества OpenCode

### 🚀 75+ провайдеров моделей
OpenCode поддерживает более 75 провайдеров через Models.dev:
- **MiMo-V2-Omni** (рекомендуется для курса)
- MiMo-V2-Pro
- Claude (все модели)
- GPT-4o
- Gemini
- DeepSeek
- Llama
- И многие другие

### 📷 Важно: выберите MiMo-V2-Omni для курса
Для прохождения курса рекомендуется использовать **MiMo-V2-Omni** — это мультимодальная модель, которая поддерживает анализ изображений. Выберите её в меню выбора модели (Ctrl+O или через селектор внизу).

### 💰 Бесплатные модели
В отличие от аналогов, OpenCode включает бесплатные модели для старта — не требуется платная подписка.

### 🖥️ Десктопное приложение
Помимо CLI, доступно полноценное десктопное приложение с графическим интерфейсом.

### 🔧 Встроенная поддержка LSP
Language Server Protocol поддерживается из коробки для автодополнения и анализа кода.

### 🤖 Переключение агентов (build/plan)
Быстрое переключение между режимами через Tab:
- **build** — генерация кода
- **plan** — планирование и анализ

### ⚙️ Гибкая конфигурация
Настройки через `~/.opencode.json` и `.opencode/` в проекте.

### 🔌 MCP поддержка
Model Context Protocol для расширения функциональности.
