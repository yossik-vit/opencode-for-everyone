# Введение в курс

## Добро пожаловать!

Научитесь использовать AI-агента прямо в VS Code — для разбора файлов, написания документов, работы с кодом и автоматизации рутины. Подключение — через корпоративный LLM Gateway AGIMA, рабочие данные не уходят в публичный сервис.

## Цели обучения

После курса вы сможете:

1. Работать с AI-агентом в боковой панели VS Code
2. Использовать AI для ускорения повседневных задач
3. Настроить `AGENTS.md` — память проекта
4. Работать с Git и GitHub с помощью AI-агента
5. Создавать и выкатывать проекты с AI
6. Настраивать кастомные команды и суб-агентов

## Пререквизиты

- Базовые знания терминала (необязательно глубокие)
- Общее понимание, что такое файлы и папки
- Git (необязательно — агент подскажет команды по ходу)

## Формат курса

### Модуль 0: Установка и настройка
- Расширение Claude Code в VS Code
- Настройка Gateway AGIMA (URL и токен)
- Первый запуск

### Модуль 1: Основы работы
- Рабочее пространство AI-агента в VS Code
- Работа с файлами и команды
- Запуск параллельных агентов
- Настройка памяти проекта (`AGENTS.md`)

### Модуль 2: Практический проект
- Планирование проекта с AI
- Итеративная разработка
- Интеграция с GitHub
- Выкатка на Vercel

### Модуль 3: Информационная безопасность
- Что можно и что нельзя отправлять AI
- Классификация чувствительных данных
- Социальная инженерия и риски ИИ

## Как проходить курс

1. **Выполняйте упражнения** — практика важнее теории.
2. **Экспериментируйте** — пробуйте свои запросы, не бойтесь сломать.
3. **Читайте `AGENTS.md`** — понимайте, как работает память проекта.
4. **Спрашивайте агента** — он умеет объяснять свои действия.

## Зачем Gateway AGIMA, а не публичный чат

- **Безопасность.** Рабочий код, тикеты, внутренние документы не уходят в чужое облако.
- **Единая точка.** Мы централизованно подключаем и обновляем модели — вы всегда на самой свежей, без отдельной подписки.
- **Контроль.** Видно совокупное потребление команды, можно выдавать и отзывать токены.

## Терминология курса

- **AI-агент** — модель, которая умеет читать файлы и выполнять действия в твоей IDE.
- **Claude Code** — расширение для VS Code (Anthropic), через которое AI-агент живёт в боковой панели.
- **Gateway AGIMA** — корпоративный прокси к языковым моделям (`https://llm-gateway.agima.tech`). Все запросы идут через него.
- **`AGENTS.md`** — файл проекта с инструкциями и памятью; агент читает его автоматически.
- **Суб-агент** — специализированный помощник (например, «ревью кода» или «тестер»), который запускается как отдельная роль.
- **MCP** — протокол расширений (Model Context Protocol), с помощью которого агент получает доступ к внешним инструментам.
