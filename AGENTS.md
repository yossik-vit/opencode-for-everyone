# OpenCode for Everyone - Course Instructions

## ⚠️ ВАЖНО: ТЫ И ЕСТЬ OpenCode!

**НЕ пытайся запустить команду `opencode`** - ты уже работаешь внутри OpenCode!

НЕ выполняй команды вроде `opencode` или `./opencode`. Это как пытаться запустить сам себя.

**Что делать:** Просто читай файлы курса и общайся с пользователем напрямую.

---

Этот репозиторий содержит адаптированный курс "OpenCode for Everyone".

### Если пользователь дал ссылку на GitHub

Если пользователь говорит "давай пройдём курс" и даёт ссылку вида `https://github.com/yossik-vit/opencode-for-everyone`:

1. **Клонируй репозиторий**: `git clone https://github.com/yossik-vit/opencode-for-everyone.git`
2. **Перейди в папку**: `cd opencode-for-everyone`  
3. **Сразу начни читать** файл `course-materials/lesson-modules/1.1-introduction/OPENCODE.md`
4. **НЕ запускай** команду `opencode` - ты уже внутри неё!
5. **НЕ спрашивай** "как хотите проходить?" - просто начинай

### Как вести курс

**НЕ спрашивай** "как хотите проходить?" или "что предпочитаете?". 

**Сразу начинай** интерактивное обучение, читая файл по секциям.

### Формат уроков

Каждый урок в `course-materials/lesson-modules/` содержит **ТОЛЬКО** файл:
- `OPENCODE.md` - адаптированный для OpenCode

**НЕТ** файлов CLAUDE.md - они удалены. Не ищи их.

### Как читать уроки

1. Найди текущий урок (начни с `1.1-introduction/OPENCODE.md`)
2. Читай файл **посекционно** - каждая секция заканчивается на `STOP:`
3. На `STOP:` - задай вопрос пользователю и жди ответа
4. После ответа - продолжай к следующей секции
5. **НЕ показывай весь файл сразу**
6. **НЕ спрашивай** "как проходить курс" - сразу начинай
