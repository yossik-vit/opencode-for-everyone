# AI-тренажёр: от 1 запроса до приложения

## Версия для AGIMA

Практический курс для сотрудников AGIMA. Учит работать с AI-агентом в IDE для повседневных задач: разбирать файлы, собирать документы, синтезировать информацию, запускать параллельных агентов, настраивать память проекта. Во втором модуле — собирается и выкатывается веб-приложение с нуля.

Курс **не требует опыта программирования**. Всё обучение — через практику, не через видео.

---

## Что нужно, чтобы начать

- **VS Code** последней версии с расширением **Claude Code** (Anthropic).
- **Доступ к LLM Gateway AGIMA** — персональный токен выдают в `#ai-platform` или у AI-чемпиона команды.
- **Git** — чтобы клонировать материалы курса.

Всё это разобрано в модуле **0.1 Установка и аутентификация** — пройди его перед началом курса.

### Шаг 1. Клонируй материалы курса

```bash
git clone https://github.com/yossik-vit/opencode-for-everyone.git ai-trainer
cd ai-trainer
code .
```

### Шаг 2. Запусти агента и начни урок

Открой панель AI-агента в VS Code (иконка слева) и напиши:

```
@course-materials/lesson-modules/1.1-introduction/CLAUDE.md давай пройдём урок
```

Агент сам прочитает инструкции, начнёт вести тебя по сценарию и будет останавливаться на вопросах.

## Сценарий курса

Курс построен вокруг вымышленной кофейни **Basecamp Coffee**. Ты получаешь «унаследованный хаос» нового менеджера — папка с разношёрстными файлами, отзывами, данными конкурентов — и на реальном сценарии осваиваешь все основные возможности AI-агента.

Навыки применимы к рабочим задачам в AGIMA: анализу проектов, созданию документации, автоматизации рутины, работе с клиентскими данными.
