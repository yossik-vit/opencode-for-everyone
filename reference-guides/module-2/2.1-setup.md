# Настройка проекта

## Создание нового проекта

### Шаг 1: Инициализация проекта

```bash
# Создайте директорию проекта
mkdir my-awesome-project
cd my-awesome-project

# Инициализируйте Git
git init

# Инициализируйте npm
npm init -y
```

### Шаг 2: Откройте проект в VS Code

```bash
# Откройте папку проекта в VS Code
code .
```

Убедитесь, что панель AI-агента (Claude Code) открыта справа — на ней будут все следующие команды.

### Шаг 3: Создание AGENTS.md

В панели агента напишите:

```
Создай файл AGENTS.md для проекта — приложение для управления задачами (todo list) на React с TypeScript.
```

Агент создаст `AGENTS.md` с описанием стека, соглашений и команд.

## Настройка стека технологий

### Установка зависимостей

```
> Установи зависимости для React проекта с TypeScript:
> - react и react-dom
> - typescript
> - vite для сборки
> - eslint и prettier для качества кода
> - vitest для тестов
```

### Настройка TypeScript

```
> Создай tsconfig.json с строгой типизацией для React проекта
```

### Настройка Vite

```
> Создай vite.config.ts с настройками для React
```

### Настройка ESLint и Prettier

```
> Создай .eslintrc.cjs и .prettierrc с настройками для TypeScript React проекта
```

## Структура проекта

### Создание директорий

```
> Создай структуру директорий:
> /src
>   /components
>   /hooks
>   /types
>   /utils
> /public
> /tests
```

### Базовые файлы

```
> Создай базовые файлы:
> - src/main.tsx (точка входа)
> - src/App.tsx (главный компонент)
> - src/index.css (глобальные стили)
> - index.html (HTML шаблон)
```

## Настройка Git

### .gitignore

```
> Создай .gitignore для Node.js/React проекта
```

### Первый коммит

```
> Добавь все файлы и сделай первый коммит с сообщением "Initial project setup"
```

## Настройка конфигурационных файлов

### package.json

```
> Обнови package.json:
> - name: "todo-app"
> - description: "React todo application"
> - Добавь скрипты: dev, build, test, lint, format
```

### Скрипты

```json
{
  "scripts": {
    "dev": "vite",
    "build": "tsc && vite build",
    "preview": "vite preview",
    "test": "vitest",
    "test:ui": "vitest --ui",
    "lint": "eslint src --ext .ts,.tsx",
    "format": "prettier --write \"src/**/*.{ts,tsx,css}\""
  }
}
```

## Настройка линтинга

### ESLint конфигурация

```
> Создай .eslintrc.cjs с правилами:
> - @typescript-eslint/recommended
> - react/recommended
> - react-hooks/recommended
```

### Husky и lint-staged (опционально)

```
> Настрой husky для запуска lint перед коммитом
```

## Настройка окружения

### Переменные окружения

```
> Создай .env.example с примерами переменных:
> - VITE_API_URL
> - VITE_APP_TITLE
```

### Типы для env

```
> Создай src/vite-env.d.ts с типами для Vite
```

## Проверка настройки

### Запуск dev сервера

```bash
npm run dev
```

### Проверка линтера

```bash
npm run lint
```

### Проверка типов

```bash
npx tsc --noEmit
```

### Запуск тестов

```bash
npm test
```

## Обновление AGENTS.md

После настройки обновите AGENTS.md:

```markdown
# Инструкции для AI-агента

## О проекте
Todo приложение на React с TypeScript.

## Стек
- React 18
- TypeScript (строгий режим)
- Vite для сборки
- Vitest для тестов
- ESLint + Prettier

## Структура
/src
  /components    # React компоненты
  /hooks         # Кастомные хуки
  /types         # TypeScript типы
  /utils         # Утилиты

## Команды
- npm run dev — dev сервер
- npm test — тесты
- npm run lint — линтер
- npm run build — сборка

## Соглашения
- Functional components с хуками
- TypeScript строгий режим
- Тесты через Vitest
- Стили через CSS modules
```
