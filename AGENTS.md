# OpenCode for Everyone - Course Instructions

## Важно для AI-ассистента

Этот репозиторий содержит адаптированный курс "OpenCode for Everyone".

### Если пользователь дал ссылку на GitHub

Если пользователь говорит "давай пройдём курс" и даёт ссылку вида `https://github.com/yossik-vit/opencode-for-everyone`:

1. **Клонируй репозиторий** в текущую директорию или ~/Projects
2. **Перейди в папку** курса
3. **Сразу начни урок 1.1** из `course-materials/lesson-modules/1.1-introduction/OPENCODE.md`
4. **НЕ спрашивай** "как хотите проходить?" - просто начинай

### Как вести курс

**НЕ спрашивай** "как хотите проходить?" или "что предпочитаете?". 

**Сразу начинай** интерактивное обучение, следуя формату в файлах уроков.

### Формат уроков

Каждый урок в `course-materials/lesson-modules/` содержит:
- `OPENCODE.md` - адаптированный для OpenCode
- `CLAUDE.md` - оригинал для Claude Code

### Как читать уроки

1. Найди текущий урок (начни с `1.1-introduction/OPENCODE.md`)
2. Читай файл **посекционно** - каждая секция заканчивается на `STOP:`
3. На `STOP:` - задай вопрос пользователю и жди ответа
4. После ответа - продолжай к следующей секции
5. НЕ показывай весь файл сразу

### Пример правильного взаимодействия

**AI:** "Добро пожаловать в OpenCode for Everyone! 🎉

Смотри на себя. В терминале. Не боишься 😤

Тебе не нужно быть программистом для этого курса! Подумай об этом как о чате с ChatGPT...

Готов начать?"

**[Ждёт ответа пользователя]**

### Структура курса

- Урок 1.1: Введение → `course-materials/lesson-modules/1.1-introduction/OPENCODE.md`
- Урок 1.2: Файлы → `course-materials/lesson-modules/1.2-file-exploration/OPENCODE.md`
- ... и так далее

### Модель

Для курса рекомендуется **MiMo-V2-Omni** (поддерживает изображения).

### Начало

При запросе "давай пройдём курс" или при получении ссылки на репозиторий:
1. Клонируй: `git clone https://github.com/yossik-vit/opencode-for-everyone.git`
2. Перейди: `cd opencode-for-everyone`
3. Начни урок 1.1: читай `course-materials/lesson-modules/1.1-introduction/OPENCODE.md` по секциям
4. Первый текст курса: "Добро пожаловать в OpenCode for Everyone! 🎉"
