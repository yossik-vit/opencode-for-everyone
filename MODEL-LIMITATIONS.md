# Ограничения модели MiMo-V2-Pro

## Что поддерживается

### ✅ Работает отлично:
- Чтение и анализ текстовых файлов
- Создание и редактирование файлов
- Поиск по файлам (grep, glob)
- Выполнение bash команд
- Веб-поиск и анализ страниц
- Создание приложений (Next.js, React, etc.)
- Git операции
- Анализ кода
- Генерация документов

### ⚠️ Частично поддерживается:
- Сложные математические вычисления
- Длинные контексты (может терять фокус)
- Очень специфичные domain knowledge

## Что НЕ поддерживается

### ❌ Не работает:
- **Анализ изображений** - MiMo-V2-Pro не может видеть картинки
- **Анализ скриншотов** - нельзя вставлять скриншоты для ревью
- **Анализ PDF с картинками** - только текстовый контент

## Как итерировать без анализа изображений

### Способ 1: Опишите словами
```
Хочу изменить цвет заголовка на синий, 
шрифт увеличить до 24px, 
добавить отступ сверху 20px
```

### Способ 2: Покажите код
```
Вот код элемента, который хочу изменить:
<div class="title">Мой заголовок</div>

Измени его так-то и так-то
```

### Способ 3: Используйте plan режим
Нажмите `Tab` для переключения в plan режим - AI покажет план изменений перед тем как их делать.

### Способ 4: Опишите проблему
```
На странице quiz результаты отображаются слишком близко друг к другу, 
нужно добавить отступы между ними
```

## Рекомендации

### Для курса:
1. **Используйте текстовые описания** вместо скриншотов
2. **Показывайте код** элементов которые хотите изменить
3. **Используйте plan режим** для сложных изменений
4. **Описывайте проблемы** словами, а не картинками

### Для разработки:
1. **Тестируйте визуально** сами в браузере
2. **Описывайте что видите** словами
3. **Используйте CSS** для точной настройки
4. **Проверяйте код** в редакторе

## Альтернативы

Если вам нужен анализ изображений, можно:

1. **Использовать другую модель** через OpenCode:
   ```json
   {
     "agents": {
       "coder": {
         "model": "claude-3.5-sonnet"
       }
     }
   }
   ```

2. **Описать изображение текстом** и попросить AI помочь на основе описания

3. **Использовать отдельные инструменты** для анализа изображений

## Настройка модели

### MiMo-V2-Pro (по умолчанию)
```json
{
  "providers": {
    "local": {
      "endpoint": "http://localhost:11434/v1",
      "apiKey": "not-needed"
    }
  },
  "agents": {
    "coder": {
      "model": "mimo-v2-pro",
      "maxTokens": 8000
    }
  }
}
```

### Claude (для анализа изображений)
```json
{
  "providers": {
    "anthropic": {
      "apiKey": "your-api-key"
    }
  },

    "coder": {
      "model": "claude-3.5-sonnet",
      "maxTokens": 8000

  }

```
