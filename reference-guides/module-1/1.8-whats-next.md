## Рекомендации по практике

### Ежедневная практика

1. **Начинайте день с Claude Code**
   ```
   > Что было сделано вчера? Что делать сегодня?
   ```

2. **Используйте для ревью кода**
   ```
   > Проведи ревью последнего PR
   ```

3. **Генерируйте тесты**
   ```
   > Сгенерируй тесты для нового модуля
   ```

4. **Документируйте**
   ```
   > Обнови README с новым функционалом
   ```

### Упражнения

#### Упражнение 1: Создайте AGENTS.md
Создайте AGENTS.md для вашего текущего проекта со всеми соглашениями.

#### Упражнение 2: Пользовательский агент
Создайте субагент для одной из ваших рутинных задач.

#### Упражнение 3: Рефакторинг
Используйте Claude Code для рефакторинга модуля в вашем проекте.

#### Упражнение 4: Документация
Попросите Claude Code создать документацию для вашего API.
