# Урок 0.1: Установка и аутентификация

Ты помогаешь сотруднику AGIMA настроить рабочее окружение для курса.
Цель: VS Code + Claude Code + LLM Gateway AGIMA — всё работает.

## Шаг 1. Проверь VS Code

Если нет — помоги скачать: https://code.visualstudio.com/download

<!-- EXPECTED: Да -->

## Шаг 2. Установи расширение Claude Code

Открой VS Code → панель расширений (Ctrl+Shift+X) → найди **Claude Code** от Anthropic → Установить.

Или одной командой в терминале:

```bash
code --install-extension anthropic.claude-code
```

После установки в левой панели появится иконка AI-агента (робот).

STOP: Расширение установлено? Видишь иконку?

<!-- EXPECTED: Да -->

---

## Шаг 3. Получи токен LLM Gateway

Для работы Claude Code через корпоративный шлюз AGIMA нужен персональный токен.

STOP: У тебя уже есть токен для LLM Gateway AGIMA?

<!-- EXPECTED: variable -->

---

### Если токена нет

Подай заявку в **сервис-деск** — тебе выдадут персональный токен и привяжут к аккаунту.

STOP: Заявка подана? Ждёшь токен?

<!-- EXPECTED: variable -->

---

Если токен уже есть — отлично, переходи к шагу 4.

## Шаг 4. Настрой settings.json

Открой палитру команд — `Cmd/Ctrl+Shift+P` — набери «Preferences: Open User Settings (JSON)».

Добавь блок:

```json
{
  "claudeCode.preferredLocation": "panel",
  "claudeCode.environmentVariables": [
    { "name": "ANTHROPIC_BASE_URL",   "value": "https://llm-gateway.agima.tech" },
    { "name": "ANTHROPIC_AUTH_TOKEN", "value": "ТВОЙ_ТОКЕН" }
  ]
}
```

Замени `ТВОЙ_ТОКЕН` на выданный тебе токен. Модель Gateway подбирает сам — указывать её вручную не нужно.

Сохрани файл. Настройки применятся сразу, перезапуск VS Code не нужен.

STOP: Настройки сохранены? Видишь JSON с gateway.agima.tech?

<!-- EXPECTED: Да -->

---

## Шаг 5. Проверь, что всё работает

Открой панель AI-агента (иконка слева). Напиши:

```
привет
```

Агент должен ответить. Если ответ пришёл — всё настроено правильно.

STOP: Агент ответил?

<!-- EXPECTED: variable -->

---

Если агент не отвечает — проверь:
1. Токен в settings.json правильный (без лишних кавычек или пробелов)
2. Нет опечаток в `ANTHROPIC_BASE_URL`
3. Напиши в **#ai-platform** — помогут разобраться
