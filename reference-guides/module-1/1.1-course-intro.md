# Введение в курс

## Добро пожаловать!

Научитесь использовать Claude Code — AI-ассистента для разработки в терминале.

## Цели обучения

После курса вы сможете:

1. Работать с Claude Code в терминале
2. Использовать AI для ускорения разработки
3. Настроить AGENTS.md
4. Работать с Git и GitHub через Claude Code
5. Создавать и развертывать проекты с AI
6. Настраивать команды и агентов

## Пререквизиты

- Базовые знания терминала
- Понимание основ программирования
- Git (базовое использование)

## Формат курса

### Модуль 0: Установка и настройка
- Установка Claude Code
- Настройка провайдеров моделей
- Первый запуск

### Модуль 1: Основы работы
- Рабочее пространство Claude Code
- Работа с файлами и команды
- Использование агентов
- Настройка памяти проекта

### Модуль 2: Практический проект
- Планирование проекта
- Итеративная разработка
- Интеграция с GitHub
- Развёртывание

## Как проходить курс

1. **Выполняйте упражнения** — практика важнее теории
2. **Экспериментируйте** — пробуйте свои запросы
3. **Читайте AGENTS.md** — понимайте работу памяти
4. **Используйте Tab** — переключайте между сборкой и планированием

## Отличия Claude Code от аналогов

| Функция | Claude Code | Другие |
|---------|----------|--------|
| Модели | 75+ провайдеров | Ограниченный выбор |
| Цена | Бесплатные модели | Подписка $20+/мес |
| LSP | Встроенный | Нет |
| Агенты | Сборка/Планирование | Нет |
| Приложение | CLI + Десктоп | Только CLI |
| Конфигурация | ~/.claude/settings.json | .env файлы |

## Терминология курса

- **Claude Code** — AI-ассистент в терминале
- **AGENTS.md** — файл инструкций и памяти проекта
- **Провайдер** — сервис AI-моделей (OpenAI, Anthropic и т.д.)
- **Агент** — режим работы (сборка или планирование)
- **MCP** — протокол контекста модели для расширений
- **LSP** — протокол языкового сервера для анализа кода
