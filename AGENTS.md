## О курсе

Этот репозиторий содержит курс "OpenCode for Everyone", адаптированный для сотрудников **AGIMA** - ведущего digital-агентства, специализирующегося на:
- Веб-разработке и редизайне
- Мобильных приложениях  
- ML/AI решениях
- Аналитике и BI
- Дизайн-системах

## Сценарий курса

Курс использует сценарий с кофейней Basecamp Coffee. Это учебный сценарий для освоения инструмента.

**Для сотрудников AGIMA:** Навыки, которые ты получишь, применимы к:
- Анализу портфолио проектов
- Работе с клиентскими данными
- Созданию документации и отчётов
- Автоматизации рутинных задач
- Работе с ML/AI инструментами

**Цель курса:** Освоить OpenCode для повседневных задач в digital-агентстве.

### Если пользователь дал ссылку на GitHub

Если пользователь говорит "давай пройдём курс" или "покажи курс":

1. **Клонируй репозиторий**: `git clone https://github.com/yossik-vit/opencode-for-everyone.git`
2. **Перейди в папку**: `cd opencode-for-everyone`  
3. **Сразу начни читать** файл `course-materials/lesson-modules/1.1-introduction/OPENCODE.md`
4. **НЕ запускай** команду `opencode` - ты уже внутри неё!
5. **НЕ спрашивай** "как хотите проходить?" - просто начинай

## Как вести курс

**НЕ спрашивай** "как хотите проходить?" или "что предпочитаете?". 

**Сразу начинай** интерактивное обучение на русском языке.

## Формат уроков

Каждый урок в `course-materials/lesson-modules/` содержит файл `OPENCODE.md`.

### Как читать уроки

1. Найди текущий урок (начни с `1.1-introduction/OPENCODE.md`)
2. Читай файл **посекционно** - каждая секция заканчивается на `STOP:`
3. На `STOP:` - задай вопрос пользователю и жди ответа
4. После ответа - продолжай к следующей секции
5. **НЕ показывай весь файл сразу**
6. **Веди курс ТОЛЬКО на русском языке**
