## Рекомендации по практике

### Ежедневно

1. **Начни день с агента:**
   ```
   Что было сделано вчера? Что в плане на сегодня?
   ```
2. **Ревью своих коммитов:**
   ```
   Сделай ревью последних изменений относительно main.
   ```
3. **Авто-генерация тестов:**
   ```
   Сгенерируй тесты для нового модуля, следуй стилю существующих тестов.
   ```
4. **Документация:**
   ```
   Обнови README — добавь новую команду и пример использования.
   ```

### Упражнения

1. **`AGENTS.md`** — создай файл для своего текущего проекта, опиши стек и соглашения.
2. **Пользовательский суб-агент** — собери «ревьюера» или «копирайтера» под свою задачу.
3. **Рефакторинг** — попроси агента переорганизовать один модуль, прогоняйте изменения через diff перед принятием.
4. **Документация API** — попроси сгенерить OpenAPI-описание по существующим эндпоинтам.
