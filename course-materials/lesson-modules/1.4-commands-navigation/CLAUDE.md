## Основные slash-команды

Введи `/` в панели агента — увидишь меню команд. Ключевые:

- **/context** — что сейчас в контексте (какие файлы агент видит).
- **/clear** — начать новый разговор с чистого листа.
- **/compact** — сжать длинный разговор, сохранив суть.
- **/usage** — сколько токенов ушло за сессию.
- **/resume** — продолжить с места остановки после перезапуска.

## Think-ключевые слова

Агент умеет думать дольше, если попросить.

- «think about X» — нормальное мышление.
- «think harder» — глубокий анализ.
- «ultrathink» — максимальная мощность (увидишь радугу!).

Используй для сложного — стратегия, аналитика, хитрые проблемы. Глубокое мышление расходует лимиты Gateway, поэтому не включай его для простых запросов.

## Три режима ввода

Три режима работы:

- **Edit mode** — по умолчанию. Агент спрашивает разрешение перед каждым действием. Безопасно, медленно.
- **Auto-accept mode** — агент просто делает. Быстро, но ты доверяешь.
- **Plan mode** — агент сначала показывает план, ты его проверяешь, потом он выполняет.

Переключение — `Shift+Tab`.

Начинай с **Edit**. Когда привыкнешь — **Auto** для простых задач. **Plan** — для больших изменений, где хочется увидеть шаги до исполнения.

## Финал: Bypass Permissions

Последний уровень — `Bypass Permissions`. Агент перестаёт спрашивать разрешение на любые действия: создаёт файлы, запускает команды, коммитит — всё без подтверждений.

⚠️ Опасно: полный доступ, никаких проверок. Используй только когда полностью доверяешь задаче.

Как включить: в панели агента внизу есть переключатель режимов. Нажми `Shift+Tab` нужное число раз, пока не появится «Bypass Permissions».

Опытные пользователи со временем работают в этом режиме для рутинных задач. Но не с первого дня.

## Завершение

**Мета-навыки:**

1. **Slash-команды** — управление контекстом, сессиями.
2. **Пользовательские команды** — ярлыки для повторяющихся действий.
3. **Прерывание и откат** — контроль, исследование без риска.
4. **Think-ключевые слова** — глубокий анализ.
5. **Режимы работы** — от Edit до Bypass Permissions.

**Где это применяется:**
- Работаешь быстро и доверяешь — Auto mode или Bypass Permissions.
- Сложная проблема — think-ключевые слова.
- Что-то не так — откатись, попробуй другой подход.
- Чистый лист — `/clear`.

## Важные заметки для AI-агента (проводящего урок)

- **Демонстрация Escape**: Начни долгую задачу, дай практиковать Escape.
- **Демонстрация отката**: Покажи Esc×2 и объясни опции.
- **Plan mode**: Не продавай сильно. Полезен для крупных изменений, но не критичен.
- **Bypass Permissions**: Упомяни как «выпускной уровень», не для первого дня.
