# Введение в курс "OpenCode для всех"

## Что такое OpenCode?

AI-ассистент для разработки в терминале. Работает с вашим кодом и проектами.

## Ключевые преимущества OpenCode

### 🚀 75+ провайдеров моделей
Поддерживает 75+ провайдеров через Models.dev:
- Claude (все модели)
- GPT-4o
- Gemini
- DeepSeek
- Llama
- и многие другие

### 📷 Важно: выберите мультимодальную модель для курса
Используйте **мультимодальную модель** с поддержкой изображений. Выберите в меню (Ctrl+O или селектор внизу).

### 💰 Бесплатные модели
Включает бесплатные модели для старта. Бесплатные модели доступны.

### 🖥️ Десктопное приложение
Полноценное десктопное приложение с графическим интерфейсом.

### 🔧 Встроенная поддержка LSP
Протокол языкового сервера для автодополнения и анализа кода.

### 🤖 Переключение агентов (сборка/планирование)
Быстрое переключение через Tab:
- **Сборка** — генерация кода
- **Планирование** — планирование и анализ

### ⚙️ Гибкая конфигурация
Настройки через `~/.opencode.json` и `.opencode/` в проекте.

### 🔌 Поддержка MCP
Протокол контекста модели для расширений.

## Для кого этот курс?

Для всех, кто хочет использовать AI в разработке:
- Начинающих разработчиков
- Опытных инженеров
- Тимлидов и архитекторов
- Product-менеджеров
- Дизайнеров
