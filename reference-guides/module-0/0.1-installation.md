# Установка OpenCode

## Системные требования

- Node.js 18+ или Go 1.21+
- macOS, Linux или Windows (WSL)
- Терминал с UTF-8

## Способ 1: Через npm (рекомендуется)

```bash
npm install -g opencode
```

## Способ 2: Через Homebrew (macOS)

```bash
brew install opencode
```

## Способ 3: Через Go

```bash
go install github.com/opencode/opencode@latest
```

## Способ 4: Десктопное приложение

Скачайте установщик с официального сайта:
- macOS (Apple Silicon / Intel)
- Windows
- Linux (AppImage / deb)

## Проверка установки

```bash
opencode --version
```

## Первоначальная настройка

### Вариант A: Бесплатные модели

OpenCode включает бесплатные модели:

```bash
opencode init
```

Ответьте "да" на вопрос об использовании бесплатных моделей.

### Вариант B: API ключи

```bash
# Установите ключ для выбранного провайдера
export OPENAI_API_KEY="sk-..."
export ANTHROPIC_API_KEY="sk-ant-..."
# или любой другой провайдер из 75+ доступных
```

### Настройка через конфигурационный файл

Создайте или отредактируйте `~/.opencode.json`:

```json
{
  "provider": "openai",
  "model": "gpt-4o",
  "providers": {
    "anthropic": {
      "apiKey": "sk-ant-..."
    },
    "openai": {
      "apiKey": "sk-..."
    }
  }
}
```

## Проверка работоспособности

```bash
# Перейдите в директорию проекта
cd my-project

# Запустите OpenCode
opencode

# Задайте первый вопрос
> Привет! Расскажи что в этом проекте
```

## Установка плагина для редактора (опционально)

OpenCode поддерживает интеграцию с популярными редакторами:

### VS Code
```bash
code --install-extension opencode.opencode-vscode
```

### Neovim (через LSP)
Добавьте в конфигурацию Neovim поддержку встроенного LSP OpenCode.

## Устранение проблем

### Проблема: Команда не найдена
Решение: Убедитесь, что путь к глобальным npm пакетам в PATH:
```bash
export PATH="$(npm root -g):$PATH"
```

### Проблема: Ошибки авторизации
Решение: Проверьте API ключи и их формат в `~/.opencode.json`.

### Проблема: Тихий вывод
Решение: Проверьте настройки терминала и кодировку UTF-8.
